# Ghi chú nghiên cứu

Thư mục này chứa các ghi chú nghiên cứu, phát hiện, và nhật ký thí nghiệm (research notes, findings, and experiment logs).

## Convention
- File name format: `YYYY-MM-DD-<chủ-đề>.md`
- Each note should include: tiêu đề, ngày, bối cảnh, nội dung chính, kết luận, câu hỏi mở
- Experiments go in `experiments/` subdirectory
- Encourage teammates to contribute their own notes
- Keep notes focused: one topic per file
